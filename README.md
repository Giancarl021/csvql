# csvql

An CLI interface to quickly query CSV data

### Demo

![Demo](docs/demo.gif)

## Installation

### CLI

npm:

```bash
npm install --global csvql
```

yarn:

```bash
yarn add global csvql
```

### API

npm:

```bash
npm install --global csvql
```

yarn:

```bash
yarn add global csvql
```
